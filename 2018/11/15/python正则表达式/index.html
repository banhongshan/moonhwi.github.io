元字符

正则表达式主要依赖于元字符. 元字符不代表他们本身的字面意思, 他们都有特殊的含义. 一些元字符写在方括号中的时候有一些特殊的意思. 以下是一些元字符的介绍:

点运算符
“.” 匹配所有的字符，除了换行符
区别于. 此字符转义后为匹配”.”符号
如果在修饰符中使用re.S之后则可以匹配包含换行符在内的所有字符

星形运算符
* 运算符作用是匹配之前的运算符，大于等于0次

？
在正则表达式中元字符?标记在符号前面的字符为可选, 即出现 0 或 1 次.
